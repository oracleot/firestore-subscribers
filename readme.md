# OnPoint Predictions
This is an app used to gather subscribers' data and send them prediction tips at intervals

## How to Use:
- Sign up on Firebase
- Create a config.js file in your scripts folder and enter your firebase configuration
- const config = { <br>
    apiKey: "", <br>
    authDomain: "", <br>
    databaseURL: "", <br>
    projectId: "", <br>
    storageBucket: "", <br>
    messagingSenderId: "" <br>
};
